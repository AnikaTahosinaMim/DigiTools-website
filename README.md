# 🚀 Digitools Platform

## 📖 Description

Digitools Platform is a web application where users can explore and use different career tools like resume builders, portfolio creators, and job toolkits in one place.

## ⚙️ Technologies Used

- React
- JavaScript
- Tailwind CSS

## ✨ Features

🛒 Add to cart system
❌ Remove tools from cart
💰 Dynamic price calculation
🏷️ Tag system (new, popular, best seller)
⚡ Fast & responsive UI
